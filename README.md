# roll20-timer-scripts
Roll20 Timer API for Digital clock
# ⏱️ Roll20 Timer & Visual FX API  
타이머 / 카운트다운 / 카운트업 표시를 위한 Roll20 전용 API

이 API는 Roll20에서 **맵 위에 실시간으로 시간이 흐르는 디지털 타이머를 표시**합니다.  
카운트다운·카운트업·특정 색상·디지털 표시 형식 등 연출에 특화되어 있으며,  
GM 명령어로 간편하게 제어할 수 있습니다.

> ⚠️ **주의:** Roll20 API는 "Pro 플랜"에서만 사용 가능합니다.

---

## ✨ 주요 기능

### 🕒 기본 타이머 기능
- 실시간 카운트다운 / 카운트업 표시
- 디지털 숫자 스타일 (00:02:15 형식)
- 타이머 시작/중지/초기화 명령어 제공
- 토큰과 연동 가능(선택사항)
1. Roll20에서 맵에 토큰을 올려놓으세요.
2. 토큰을 더블 클릭해서 속성창을 엽니다.
3. 이름(Name) 항목에 타이머에 사용할 이름을 입력합니다.
👉 예: TurnClock
이렇게 설정해 두면 토큰 위에 타이머가 뜹니다. 

### 🎯 GM 권한 전용 제어
- GM만 명령을 실행할 수 있도록 보호
- 플레이어는 타이머를 조작할 수 없음

---

## 📦 설치 방법 (Roll20 내부)

1. Roll20 → 캠페인 열기  
2. 왼쪽 상단 메뉴 → **Settings → API Scripts**  
3. 기존 스크립트가 있다면 “New Script” 클릭  
4. 파일 이름 입력: `timer.js`  
5. 타이머 API 코드 전체 붙여넣기  
6. **Save Script** 클릭  
7. API Output Console에 오류가 없는지 확인

---

# 💻 명령어 리스트 (GM 전용)

아래 명령들은 Roll20 채팅창에 입력합니다.

| 명령어                       | 설명                             |
| ---------------------------- | ------------------------------ |
| `!timer start 3 0 TurnClock` | “TurnClock” 토큰 위에서 3분 카운트다운 시작 |
| `!timer stop`                | 타이머 일시정지                       |
| `!timer reset`               | 텍스트 오브젝트 제거 및 초기화              |

# 💻 커스터마이즈  (GM 전용)

| 수정 항목  | 위치                               | 설명                 |
| ------ | -------------------------------- | ------------------ |
| 폰트 크기  | `font_size: 48`                  | 더 크게 / 작게 조정 가능    |
| 색상     | `color: '#ffffff'`               | 기본색 (흰색) 변경 가능     |
| 깜박임 시점 | `if (timerData.remaining <= 30)` | 30초 대신 10초 등으로 바꾸기 |
| 깜박임 색상 | `flickerColor = '#ff0000'`       | 붉은색 대신 다른 색 지정     |
